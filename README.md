# f1_game_python
Simple racing game where the cars have to dodge obstacles

    Two f1 cars, a track and a lot of obstacles to dodge.
    
    Whoever collides with obstacles fewer times, wins the race!


    COMMANDS:
        
        PLAYER A: 'w' =         Up
                  's' =         Down
                  'a' =         Left
                  'd' =         Right
                  'space' =     Acceleration
                  'q' =         Deceleration
             
        PLAYER B: 'up arrow' =      Up
                  'down arrow' =    Down
                  'left arrow' =    Left
                  'right arrow' =   Right
                  '0' =             Acceleration
                  '1' =             Deceleration
